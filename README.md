# 🧠 Frontend Quiz App

A responsive and interactive quiz application built using **React**, **Context API**, and **Tailwind CSS**. Users can select a subject, answer multiple-choice questions, and receive instant feedback with a score at the end.

---

## 🚀 Features

- 📚 Multiple subjects (HTML, CSS, JavaScript, Accessibility)
- ✅ Select and submit answers
- 🎯 Instant correct/wrong feedback
- 📊 Progress bar (updates after submission)
- 🌙 Dark / Light mode toggle
- 🔄 Next question navigation
- 🧮 Score tracking
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- **React** (Functional Components + Hooks)
- **React Router DOM** (Routing)
- **Context API** (Global state management)
- **Tailwind CSS** (Styling)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/ShivaniCharan4101997/quizApp
```

2. Navigate to project folder:

```
cd quiz-app
```

3. Install dependencies:

```
npm install
```

4. Start development server:

```
npm run dev
```

---

## 🎮 How It Works

1. Choose a subject
2. Select an answer
3. Click **Submit Answer**
4. Get feedback (green/red)
5. Click **Next Question**
6. View final score
